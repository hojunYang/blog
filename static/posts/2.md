# SvelteKit으로 풀스택 앱 만들기

SvelteKit은 Svelte 기반의 풀스택 프레임워크입니다.

## 서버 사이드 렌더링

SvelteKit은 기본적으로 SSR을 지원합니다. 이를 통해 SEO와 초기 로딩 성능을 개선할 수 있습니다.

## API 라우트

`+server.ts` 파일을 통해 API 엔드포인트를 쉽게 만들 수 있습니다.

```typescript
export async function GET() {
  return new Response(JSON.stringify({ message: 'Hello' }));
}
```

## 폼 액션

SvelteKit의 폼 액션을 사용하면 JavaScript 없이도 폼을 처리할 수 있습니다.

## 결론

SvelteKit은 현대적인 웹 애플리케이션 개발을 위한 완벽한 도구입니다.
