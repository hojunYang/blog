# 웹 성능 최적화 가이드

웹 성능은 사용자 경험과 SEO에 직접적인 영향을 미칩니다.

## 이미지 최적화

1. **적절한 포맷 사용**: WebP, AVIF 등 현대적인 이미지 포맷을 사용하세요.
2. **Lazy Loading**: 화면에 보이는 이미지만 로드합니다.
3. **반응형 이미지**: `srcset`을 사용하여 디바이스에 맞는 크기의 이미지를 제공합니다.

## 코드 스플리팅

필요한 코드만 로드하여 초기 번들 크기를 줄입니다.

```typescript
const Component = lazy(() => import('./Component.svelte'));
```

## 캐싱 전략

적절한 캐싱 헤더를 설정하여 재방문 시 로딩 속도를 개선합니다.

## CDN 활용

정적 자산을 CDN을 통해 제공하여 전 세계 사용자에게 빠른 속도를 제공합니다.

## 결론

작은 최적화들이 모여 큰 차이를 만듭니다.
