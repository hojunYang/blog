# Svelte 5의 새로운 기능들

Svelte 5는 완전히 새로운 반응성 시스템을 도입했습니다.

![read-image](/images/image-01.jpg)
## Runes란?

Runes는 Svelte 5에서 도입된 새로운 반응성 프리미티브입니다. `$state`, `$derived`, `$effect` 등의 runes를 통해 더 명확하고 강력한 반응성 관리가 가능해졌습니다.

### $state

```typescript
let count = $state(0);
```

### $derived

```typescript
let doubled = $derived(count * 2);
```

### $effect

```typescript
$effect(() => {
  console.log('Count changed:', count);
});
```

## 결론

Svelte 5의 새로운 기능들은 개발자 경험을 크게 향상시켰습니다.
